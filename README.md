# Futuristic Data Science Portfolio

A premium, futuristic portfolio for a B.Tech CSE student specializing in Data Science and Machine Learning.

## Stack

- React + Vite
- Tailwind CSS
- Framer Motion
- GSAP with ScrollTrigger
- Three.js via React Three Fiber

## Run locally

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
pnpm preview
```
